# PONG.ASM
## A Pong clone for MS-DOS built using the TASM assembler
### by @DavidKaluta
A Pong game built 100% in *assembly*.
Press Esc to leave to the MS-DOS prompt.
W/S control left paddle.
A/D control right paddle.
